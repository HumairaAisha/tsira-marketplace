# Tsira | Social Marketplace

## Overview
Tsira is a modern social marketplace that blends the familiarity of social media with the functionality of e-commerce.
This project is a landing page designed to introduce the Tsira brand, communicate its value, and guide users toward engagement.

## Mission
To simplify online buying and selling while promoting trust, transparency, and eco-friendly practices.

## Features
- Social Marketplace Experience
Discover products through a familiar, scroll-based interface.
- Secure Transactions
Built with safety in mind to reduce fraud and improve trust.
- Integrated Wallet System
Seamless payments without relying on external platforms.
- Eco-Friendly Focus
Encourages reuse, resale, and sustainable buying habits.
- Smooth Buyer–Seller Interaction
Designed to create transparent and peaceful transactions.

## Design System
Tsira uses a custom color system built with Tailwind v4 theme tokens:

- Primary: `#1E5EFF`
- Secondary: `#94A3B8`
- Background: Dark-based UI for contrast and focus
- Typography: Clean, readable, and modern
